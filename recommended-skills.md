# Recommended Skills

## Core workflow

- `using-agent-skills` — discover and invoke the right workflow skill.
- `context-engineering` — load the right context at the right time.
- `spec-driven-development` — define requirements before coding.
- `incremental-implementation` — build in thin vertical slices.
- `test-driven-development` — failing test first, then make it pass.

## Code quality

- `code-review-and-quality` — multi-axis review before merging.
- `security-and-hardening` — OWASP prevention and input validation.
- `code-simplification` — simplify code while preserving behavior.
- `frontend-ui-engineering` — production-quality UI construction.

## Git and collaboration

- `git-workflow-and-versioning` — clean commits and history.
- `documentation-and-adrs` — record decisions and documentation.

## Planning and debugging

- `planning-and-task-breakdown` — decompose work into ordered tasks.
- `debugging-and-error-recovery` — systematic root-cause debugging.
