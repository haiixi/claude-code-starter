# Recommended Plugins

Install these from the official marketplace with `/plugin install <name>@claude-plugins-official`.

## Git and collaboration

- `commit-commands@claude-plugins-official` — `/commit` and `/commit-push-pr` workflows.
- `pr-review-toolkit@claude-plugins-official` — specialized PR review agents.

## Code quality

- `code-review@claude-plugins-official` — automated code review.
- `code-simplifier@claude-plugins-official` — simplify code while preserving behavior.

## Frontend and feature development

- `frontend-design@claude-plugins-official` — polished, distinctive UI components.
- `feature-dev@claude-plugins-official` — end-to-end feature development workflow.

## Automation

- `hookify@claude-plugins-official` — create automation rules from conversation patterns.
